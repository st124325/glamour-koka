ПАПКА ДЛЯ ИЗОБРАЖЕНИЙ
=====================

Сюда нужно положить следующие изображения:

ОБЯЗАТЕЛЬНЫЕ:
- logo.png          - логотип (рекомендуется 200x200 px, PNG с прозрачностью)
- avatar.jpg        - фото для секции "Обо мне" (рекомендуется 800x1000 px)
- hero-bg.jpg       - фоновое изображение для главной секции (1920x1080 px минимум)

ОБЛОЖКИ БИТОВ:
- beat-cover-1.jpg  - обложка для бита 1 (квадратная, 500x500 px)
- beat-cover-2.jpg  - обложка для бита 2
- beat-cover-3.jpg  - обложка для бита 3
- и так далее...

Рекомендации:
- Формат: JPG или PNG
- Обложки битов должны быть квадратными
- Используйте сжатие для оптимизации загрузки
- Тёмные изображения лучше сочетаются с дизайном сайта
